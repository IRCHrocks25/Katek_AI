# Images Directory

Place all your image assets in this directory.

## Required Images

See `IMAGE_PLACEHOLDERS.md` in the project root for a complete list of required images and their specifications.

## Quick Reference

- `hero-background.jpg` - Hero section background
- `hero-dashboard.jpg` - Dashboard mockup screenshot
- `website-mockup.jpg` - Website preview
- `landing-page-preview.jpg` - Landing page preview
- `reputation-chart.jpg` - Analytics chart
- `testimonial-1.jpg` through `testimonial-4.jpg` - Profile photos

All images will automatically load when placed here with the correct filenames.
